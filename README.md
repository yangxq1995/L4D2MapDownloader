## ✨ 功能特性

### 核心功能
- 🔍 **智能搜索** — 支持中/英文地图名称搜索，模糊匹配
- ⬇️ **多线程下载** — 最多 3 个地图同时下载，提升效率
- 📊 **动态超时** — 根据地图大小自动调整下载超时时间（每 100MB 给 5 分钟）
- 🔄 **自动重命名** — 下载完成后自动重命名为 `[战役]地图名.vpk` 格式
- ✅ **文件校验** — 检查本地文件大小，避免重复下载
- 📂 **游戏适配** — 自动处理 `.vpk` 和 `_legacy.bin` 文件

### 界面功能
- 📝 **可视化进度条** — 实时显示下载进度
- ☑️ **全选/取消全选** — 快捷操作按钮
- 🔘 **右键菜单** — 快速访问 Steam 创意工坊、gameMaps、B站实况
- 📊 **日志过滤** — 支持按级别过滤日志（全部/信息/警告/错误）
- 📏 **自适应布局** — 表格列宽自动调整，窗口大小可拖拽
- 🎨 **绿色主题** — 白底绿按钮的清爽界面

### 高级功能
- 🌐 **gameMaps 备用** — 无 Steam 工坊版本的地图自动打开 gameMaps 下载页面
- 📺 **B站实况** — 一键搜索该地图的 B站实况视频
- 🧹 **自动清理** — 保留最近 10 个日志文件，下载完成后清理 SteamCMD 缓存
- 💾 **配置持久化** — 自动保存窗口位置、大小和搜索历史

## 🚀 使用方法

### 1. 启动程序

```bash
python L4D2MapDownloader.py
```

或双击已打包的 `L4D2MapDownloader v2.0.exe`（如果已生成）。

### 2. 搜索地图

1. 在搜索框输入地图名称（支持中文/英文）
2. 点击 **🔍 搜索** 按钮或按 **Enter** 键
3. 等待搜索结果（从数据库查询）

### 3. 下载地图

1. 在表格中选中要下载的地图（可多选）
2. 点击 **⬇ 下载选中** 按钮
3. 等待下载完成（进度条显示状态）

### 4. 右键菜单

选中地图后右键，可快速访问：
- **观看实况** — 在 B站搜索该地图的实况视频
- **查看创意工坊** — 打开 Steam 创意工坊页面
- **Steam下载** — 重新下载该地图
- **gameMaps下载** — 打开 gameMaps.com 下载页面

## 🗄️ 数据库结构

程序从 `mapsinfo` 表查询地图信息，表结构如下：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| **modID** | varchar | Steam 工坊 ID |
| **mapNameCN** | varchar | 地图中文名称 |
| **mapNameEN** | varchar | 地图英文名称 |
| **mapAuthor** | varchar | 地图作者 |
| **mapSize** | float | 地图大小（MB） |
| **updateTime** | datetime | 最近更新时间 |
| **isUsing** | tinyint | 是否可用（0=停用，1=正常） |
| **isValue** | tinyint | 是否跳过（1=不显示） |
| **gamemapsID** | varchar | gameMaps.com 的地图 ID |

## ❓ 常见问题

### 1. SteamCMD 下载失败？

**可能原因**：
- 网络问题（Steam 服务器连接超时）
- 地图已从创意工坊删除
- SteamCMD 未正确安装

**解决方法**：
- 检查网络连接（是否需要代理）
- 手动打开创意工坊页面确认地图是否存在
- 删除 `steamcmd/` 目录，让程序重新下载

### 2. 下载的地图文件大小异常？

**判断标准**：
- 如果文件大小 < 10MB，程序会自动删除并提示
- 可能是 SteamCMD 下载不完整

**解决方法**：
- 重新下载该地图
- 如果多次失败，使用 gameMaps 备用链接下载

### 3. 数据库查询失败？

**可能原因**：
- 数据库服务器无法访问
- 缺少 `pymysql` 依赖

**解决方法**：
```bash
pip install pymysql
```

### 4. 程序卡死或无响应？

**可能原因**：
- SteamCMD 下载超时（默认动态超时）
- 多线程下载冲突

**解决方法**：
- 关闭程序，删除 `steamcmd/downloads/` 目录
- 重新打开程序，分批下载地图

### 5. 如何打包成 exe？

使用 `pyinstaller`：

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --icon=download.ico L4D2MapDownloader.py
```
